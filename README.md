# Fragmenter 1.0

Fragmenter is a Windows-first research, project-management, and game-setup
workbench for .hack//frägment

## Current cut — OPERATION DRAGONEGG

The source launcher and packaged Windows build share one first-run presentation,
asset path, cancellation policy, and in-process 3D runtime.

## Start Fragmenter

From a clean checkout, double-click:

```bat
START_FRAGMENTER_PUBLIC.bat
```

Python 3 with Tkinter is required when running from source. The packaged
`Fragmenter.exe` does not require Python.

## Game Setup

**Game Setup** collects the playable-game functions in one place:

- Build **Fragment 4.0 English** directly from the untouched Japanese ISO.
- Configure the PCSX2 keyboard and network settings with an INI backup.
- Install or inspect the supplied network-ready memory card.
- Back up and restore Area Server saves and memory cards.
