# Fragmenter application branding

The Fragmenter mark in this folder is derived from the Serenial dragon artwork supplied by Fade Dragontear for this application.

- `Fragmenter-Serenial.png` is the transparent 256 px runtime/window mark.
- `Fragmenter.ico` contains 256, 128, 64, 48, 32, and 16 px Windows icon layers.

The square framing and icon sizes are mechanical derivatives of the supplied artwork. No third-party branding is included.
